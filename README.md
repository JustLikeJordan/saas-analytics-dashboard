# SaaS Analytics Dashboard

Full-stack SaaS dashboard with auth, CSV analytics, charts, Stripe-based Pro tier, and an AI-powered smart summary.

## Problem

Small businesses need affordable analytics without paying for enterprise tools.

## Tech Stack (Planned)

- Frontend: Next.js (TypeScript), Chart.js or Recharts
- Backend: Node/Express (or Next API routes)
- Database: PostgreSQL (Supabase in production)
- Auth: JWT + Google OAuth
- Payments: Stripe
- AI: LLM API for smart summary
